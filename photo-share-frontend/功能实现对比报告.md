## 技术架构对比

### 前端技术栈
- **框架：** Vue 3 + Composition API
- **UI组件：** Vant 4（移动端组件库）
- **状态管理：** Pinia
- **路由：** Vue Router 4
- **HTTP客户端：** Axios
- **构建工具：** Vite

### 后端技术栈
- **运行环境：** Node.js
- **框架：** Express.js
- **数据库：** MySQL
- **ORM：** Sequelize
- **认证：** JWT
- **文件上传：** Multer
- **跨域：** CORS

## 数据库设计

### 核心表结构
1. **users** - 用户表：id, phone, password, nickname, avatar, status
2. **photos** - 图片表：id, userId, title, description, imageUrl, tags, likes, comments
3. **comments** - 评论表：id, userId, photoId, content, parentId, replyToUserId
4. **likes** - 点赞表：id, userId, targetType, targetId
5. **follows** - 关注表：id, followerId, followingId

### 关联关系
- 用户与图片：一对多
- 用户与评论：一对多
- 图片与评论：一对多
- 评论自关联：回复关系
- 用户与点赞：多对多
- 用户与关注：多对多
